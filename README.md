# rxmos-skills

High-quality agent skills focused on safe execution and production readiness.

## Install

```bash
npx skills add <your-github-user>/rxmos-skills --skill refactor-plan-pro
```

## Available Skills

- `refactor-plan-pro`: Creates a phased, verifiable refactor plan with risk, rollout, and rollback guidance.

## Local validation

```bash
npx skills add . --list
```
